# 🌾 Gram Sabha Balua - Village Management System

A modern and responsive web application designed for digital village management.

## 🚀 Features
- 🌐 Multi-language support (Hindi & English)
- 📢 Announcement system
- 📅 Event management UI
- 👑 Gram Pradhan profile section
- 🖼️ Image gallery with animations
- 🎨 Dark/Light mode support
- ⚡ Smooth animations using Framer Motion

## 🛠️ Tech Stack
- React.js
- Tailwind CSS
- Redux Toolkit
- Framer Motion

## 📌 Goal
To bring digital transparency and accessibility to village-level management systems.
